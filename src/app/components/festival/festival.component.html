Festival de Música

Trabajo en el que se usa Bootstrap y Sass para dar estilos a los componentes del sitio web.
Las características principales de este proyecto son:
Uso de Stylelint para mejorar la unidad de los estilos usados (unidades métricas, nombres de las clases, etc.)
Bootstrap para la creación de componentes como Card, Carousel, Navbar, etc.
Uso de Sass, procesador CSS (variables, anidado, funciones, parciales e importación)
