La casa de papel

Algunas de las características que este sitio web presenta son:

  • Creación de SVG con la herramienta Inkscape.
  • Responsive images: Resolution switching y dirección de arte.
  • Uso de clip-path, animaciones y transiciones.
  • Uso de formatos PNG, WEBP y JPG.
  • Lazy loading.

A continuación, se muestra el baremo de rendimiento tanto en versión móvil como en versión ordenador. Este se ha realizado con el uso de la herramienta Page Speed Insights (conexión Regular 3G). Para hacer la media se han descartado los valores más extremos en cada tabla.

  • Puntuación = P
  • First Contentful Paint = FCP
  • Time to Interactive = TI
  • Speed Index = SI
  • Total Blocking Time = TBT
  • Largest Contentful Paint = LCP
  • Cumulative Layout Shift = CLS
