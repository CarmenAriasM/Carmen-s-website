Aer Iolar
Trabajo de fin de máster donde se ha creado el sitio web de una aerolínea ficticia llamada Aer Iolar. Desplegado en Netlify con el backend en Heroku. Para crear esta aplicación web progresiva (PWA), se ha usado Angular para el frontend y Laravel para el backend.
En cuanto al diseño de este proyecto, se ha hecho un estudio de usabilidad, asimismo, también se ha creado un perfil de usuario y prototipos tanto en versión móvil como en versión ordenador en Figma. Si le interesa ver el trabajo escrito completo, con los prototipos incluidos, puede descargarlo aquí.

Algunas de las funcionalidades que este sitio web presenta son:
  • Búsqueda de vuelos.
  • Gestión de reservas mediante búsqueda de número de referencia.
  • Registro e inicio de sesión.
  • Recuperación de contraseña mediante correo electrónico.
  • Actualización del perfil de usuario.
  • Mapa de asientos interactivo.
  • Envío de formulario de ayuda al correo de la empresa.
